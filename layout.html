Contoh Layout dengan Table

Halaman ini mendemonstrasikan penggunaan tag <table> HTML untuk membuat layout website. Layout ini terdiri dari 4 komponen utama:

Header

Bagian atas halaman yang berisi logo, nama, dan identitas website.

Sidebar

Panel navigasi samping yang berisi link menu dan informasi singkat.

Main Content

Area konten utama yang menampilkan informasi penting halaman.

Footer

Bagian bawah halaman yang berisi copyright dan informasi tambahan.

Catatan: Meskipun tag <table> dapat digunakan untuk layout, praktik modern web development merekomendasikan penggunaan CSS Flexbox atau CSS Grid untuk layout. Table sebaiknya digunakan untuk data tabular.
